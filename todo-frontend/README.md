# Dotnet React Todo App

This project is a full-stack To-do List application featuring:

## Backend (.NET Web API)
- RESTful API endpoints: Create, Read, Update, Delete todos.
- Input validation to prevent empty todos.
- In-memory data storage (can be replaced with SQLite or other DB).
- Swagger (OpenAPI) documentation for easy testing.
- Clean Architecture structure (Core, Application, Infrastructure, API layers).
- Ready for unit testing of business logic.

## Frontend (React)
- Display all todos.
- Add, update, delete, and mark todos as completed.
- Connected to the backend API via HTTP requests.
- Simple and responsive user interface for easy interaction.
